# About My Website

This is a personal website built using Chirpy - a minimal, sidebar, responsive web design Jekyll theme that focuses on text presentation. 
[Live Demo Â»](https://chirpy.cotes.info)

## Features

- Pinned Posts
- Configurable theme mode
- Double-level Categories
- Last modified date for posts
- Table of Contents
- Automatically recommend related posts
- Syntax highlighting
- Mathematical expressions
- Mermaid diagram & flowchart
- Search
- Atom Feeds
- Disqus Comments
- Google Analytics
- GA Pageviews reporting (Advanced)
- SEO and Performance Optimization

### Documentation
